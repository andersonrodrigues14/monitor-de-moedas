# Dashbord de Moedas
Esse projeto é um aplicativo que busca em tempo real a variação do preço de moedas dos últimos 15 dias em comparação com o real, utilizando Ruby on Rails.

## Tecnologias
- Ruby
- Net/http
- Json
- chartkick
